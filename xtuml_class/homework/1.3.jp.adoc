== Account Creation
左記にアカウントを作りなさい [github.com](https://github.com)

(github.com にすでにアカウントを持っているならばそれを使ってもよいです。)

* 左記のリポジトリをフォークします https://github.com/xtuml/models[xtuml/models]
  ** 下記にてハイライトされたボタンをクリック:
     image:../img/fork_button.png[fork button]

== GitHub にログイン
Open https://github.com/&lt;your_username&gt;/models

* 上記のURLにある "&lt;your_username&gt;" をあなたのGitHub.com ユーザ名に置き換えなさい。
* このWebページのスクリーンショットを取りなさい。

== Development Workspace
自分用の開発ワークスペースを作る

* あなたのフォークにある `models` リポジトリをローカルにクローンしなさい。

([video tutorial](https://youtu.be/jkMbvtbbCLA)).

== プロジェクトのインポート
あなたのローカル モデル リポジトリから相撲モデルをインポートしなさい。

(https://youtu.be/a0AD57W-jOk[video tutorial])

* モデル リポジトリから`Sumo`, `Sumo Simulator`, `MCLMShared` をインポート
* インポートしたBridgePoint アプリケーションと共に写真を撮る。
