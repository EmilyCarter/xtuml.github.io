== アカウント作成
ここにアカウントを作成する [github.com](https://github.com)

(もし、すでに　github.com　にアカウントを持っているならば、それを使いなさい。)

* 左記を自分用にフォークする https://github.com/xtuml/models[xtuml/models] repository.
  **下記にある赤くハイライトされたものをクリック
     image:../img/fork_button.png[fork button]

== GitHubにログイン
左記を開く https://github.com/&lt;your_username&gt;/models

* 自分のユーザ名のURLに書き換え "&lt;your_username&gt;"
* このWebページの画面をコピー

== 開発ワークスペース
自分用のワークスペースを確保する

* 自分のフォークのモデルを複製してローカルディスクに置く。

([video tutorial](https://youtu.be/jkMbvtbbCLA)).

== プロジェクトのインポート
相撲(sumo)プロジェクトを自分のローカル・リポジトリにインポートする。

(https://youtu.be/a0AD57W-jOk[video tutorial])

* モデル・リポジトリから、左記をインポートする。`Sumo`, `Sumo Simulator`, `MCLMShared`
* BridgePoint 画面で Sumo プロジェクトがインポートされたようすを画面コピー
