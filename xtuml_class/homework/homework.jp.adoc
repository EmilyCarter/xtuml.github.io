=== 事前成績

学生は、事前成績を取るために定められた日時までにそれぞれの宿題を提出すること。
事前成績は成績簿には記録しません。　講師は、時間の許す限り提出された宿題を見て
学生諸君にフィードバックを行います。
宿題が完璧であれば、講師はそれが完璧であると伝えます。もしも早めに提出された宿題に
何か問題があるばあい、講師は手始めに最初の問題を学生に伝えるでしょう。
学生は、それを改訂して通常の提出日に出してください。

事前成績をとるために、提出期限よりまる一日は早く提出をしてください。
