== Email
担当の{instructor}先生にｅメールを送りなさい。 メールアドレス： __{instructoremail}__.
メール本文には、以下を書いて送ってください：

* 名前
* 写真 (自撮り可)
* 自分の研究課題
* 授業で学びたいこと

担当の先生から返信が来ます。
先生には、いつでもメールしてかまいません。英語の苦手な人は日本語でもよいです。
