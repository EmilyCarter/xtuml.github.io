Goal of this lab

The goal of this lab is to convert this web page to a document looks as close as possible to this web page using a word processor. You may use any word processor (Google Docs or Microsoft Word or similar). Do this as fast as you can. The first person to get the page converted wins. You may do this however you want, but if you don't know where to start, follow the steps below.

Step 1: View the source

  • Right click this page
  • Select "View page source"

Step 2: Open your word processor

  1. Open your word processor program (or go to Google Docs in your browser)
  2. Create a new document
  3. Use the formatting tools in the word processor to make the document look like the web page. Use the web page in your browser and the page source as a reference
